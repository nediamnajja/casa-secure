# Casa Secure

## Project Overview
**Casa Secure** is a smart home security system that allows users to monitor, control, and manage the security of their homes remotely. The system integrates sensors, cameras, and automated alerts to provide real-time notifications and control through a user-friendly interface. It is designed to enhance home safety, reduce response times in emergencies, and offer peace of mind to homeowners.

## Features
- **Real-time Monitoring:** Monitor doors, windows, and motion sensors remotely.
- **Smart Alerts:** Receive instant notifications on unusual activities via app or email.
- **Automated Control:** Control lights, locks, and alarms based on sensor triggers.
- **Data Logging:** Logs sensor events and system activities for review and analysis.
- **User Management:** Secure login and role-based access for multiple users.
- **Integration:** Supports integration with smart home devices and IoT platforms.

## Technologies Used
- **Backend:** `Java` (JavaFX for GUI), MQTT for device communication
- **Frontend:** JavaFX interface for desktop monitoring
- **Database:** SQLite/MySQL for storing user and event data
- **Protocols:** MQTT or HTTP for device communication and alerts
- **Other:** JSON for configuration and event logging
